# Karpathy-Aligned Project

A project whose Behavioral Guardrails align with Karpathy principles.

## Behavioral Guardrails

### Think Before Coding

Surface assumptions before writing code. When uncertain, ask.

### Simplicity First

Solve the stated problem with the minimum that works.

### Surgical Changes

Touch only what the user asked for.

### Verification, not Specification

Frame work as a verifiable outcome.
