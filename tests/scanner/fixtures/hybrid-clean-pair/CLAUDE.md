## 🧠 Behavioral Guardrails

When editing source files:

  1. **Think Before Coding**   →  surface assumptions; reject sycophancy
  2. **Simplicity First**       →  no overengineering; minimum code
  3. **Surgical Changes**       →  every line traces to the request
  4. **Verification, not Specification**  →  declarative outcomes
